# Changelog

## 0.3.0 - 2026-07-07

- Adiciona Data Quality Monitor dedicado com score, completude por coluna, severidade e recomendacoes.
- Adiciona Centro de Logs Operacionais com filtros por automacao, nivel e busca textual.
- Adiciona endpoints `/api/data-quality/analyze`, `/api/logs`, `/api/runs`, `/api/runs/[id]` e `/api/insights`.
- Mantem compatibilidade com a arquitetura Next.js 16 existente e expande cobertura de testes.

## 0.1.0 - 2026-06-01

- Cria o Python Automation Center com Next.js App Router.
- Adiciona cinco automacoes simuladas.
- Implementa execucao com progresso, timeline, logs e downloads.
- Adiciona dashboard executivo com Recharts.
- Adiciona relatorios, dados mockados, scripts Python e documentacao.
- Configura testes com Vitest e validacoes de build.

## 0.2.0 - 2026-06-01

- Adiciona evolucao senior com 8 automacoes simuladas.
- Adiciona Automation Marketplace com filtros avancados.
- Adiciona Rule Engine, Queue Simulator, Data Diff, Data Quality Score e Audit Trail.
- Adiciona Execution History, Architecture e Case Study.
- Adiciona System Health Panel, Risk Panel, endpoint `/api/health` e `/api/executions`.
- Adiciona script `scripts/generate_reports.py`.
- Amplia testes de API, rule engine e qualidade de dados.
